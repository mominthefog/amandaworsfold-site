This folder contains reusable context artifacts for LLM chats, Notion, and other projects.

## Files

| File | Purpose |
|------|---------|
| **COMPLETE-KNOWLEDGE-BASE.md** | Single-file comprehensive reference (use for quick copy-paste) |
| **POSITIONING.md** | Brand identity, taglines, credentials, philosophy, origin story |
| **SERVICES.md** | Full services catalog with pricing, descriptions, and deliverables |
| **TARGET-AUDIENCE.md** | ICP profiles, pain points, anti-personas, customer journey |
| **VOICE-AND-MESSAGING.md** | Tone guidelines, signature phrases, copy patterns, visual brand |
| **LLM-CONTEXT-PROMPT.md** | Ready-to-use system prompts for AI tools (full + condensed versions) |

## How to Use

**For Notion:**
Copy relevant files into your Notion workspace. Tag appropriately for search.

**For LLM Chats:**
Paste the system prompt from `LLM-CONTEXT-PROMPT.md` at the start of conversations, or use `COMPLETE-KNOWLEDGE-BASE.md` as comprehensive context.

**For Content Creation:**
Reference `VOICE-AND-MESSAGING.md` for tone and patterns. Use `SERVICES.md` for accurate pricing/descriptions.

**For Sales/Marketing:**
Use `TARGET-AUDIENCE.md` to understand who you're speaking to and their pain points.

## Keeping Updated

When services, pricing, or positioning changes, update the relevant individual files AND the `COMPLETE-KNOWLEDGE-BASE.md` combined file.
