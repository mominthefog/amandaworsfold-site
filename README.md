# Amanda Worsfold

Personal website for Amanda Worsfold's AI strategy and implementation consulting practice.

## About

This site showcases **The Knowledge Build** — a service that helps growing businesses turn scattered knowledge into AI-powered systems they can actually talk to.

## Structure

- `index.html` — Homepage
- `the-knowledge-build.html` — Service details
- `about.html` — About Amanda
- `insights.html` — Blog/articles
- `contact.html` — Contact page
- `styles.css` — Site styles
- `main.js` — Navigation and interactions

## Local Development

Open `index.html` in a browser, or use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Tech Stack

- Static HTML/CSS/JS
- Google Fonts (Cormorant Garamond, Montserrat)
- No build process required
